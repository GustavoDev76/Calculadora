# Calculadora em Python

Este é um projeto simples de calculadora desenvolvido em Python como exercício do curso de Programação do Senac.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de divisão por zero

## Tecnologias

- Python 3

## Como executar

1. Baixe o projeto.
2. Abra o terminal na pasta do projeto.
3. Execute:

```bash
python calculadora.py
```
